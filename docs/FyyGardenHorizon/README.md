## Overview

`scripts/FyyGardenHorizon.lua` targets Garden Horizon workflows. It uses the same Fyy WindUI window pattern, config persistence helpers, and player-quality-of-life style found in the other Fyy scripts.

## Feature Areas

| Area | Features |
|---|---|
| UI | `Fyy X Garden Horizon` window, AMOLED theme, keyboard toggle |
| Config | save/load/autoload helpers and local config file support |
| Farming | Garden Horizon oriented automation controls |
| Player | movement and utility helpers for smoother farming workflows |
| Notifications | user feedback through WindUI notifications |

## Runtime Model

```text
load WindUI
  -> create Garden Horizon window
  -> prepare config storage
  -> build feature controls
  -> run farming/player callbacks
```

## Best Use Case

Use this script for the Garden/farming side of the Fyy Community toolkit.
