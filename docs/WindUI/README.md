## Overview

`lib/WindUI.lua` is the local UI library dependency used by several scripts in this repository. It provides the window, theme, tab, control, notification, and visual styling primitives for WindUI-based Fyy releases.

## Provides

| Area | Features |
|---|---|
| Windows | main window creation and topbar behavior |
| Themes | color themes and custom theme values |
| Tabs | tabbed navigation for script feature groups |
| Controls | buttons, toggles, sliders, dropdowns, inputs, and related elements |
| Notifications | toast-style status and error feedback |
| Styling | icons, colors, fonts, transparency, and layout helpers |

## Used By

- `scripts/FyyAbyss.lua`
- `scripts/FyyFishITWindUI.lua`
- `scripts/FyyGardenHorizon.lua`

## Best Use Case

Keep this vendored if release stability is more important than always pulling the newest remote UI build.

## Privacy Notes

Official library attribution can be documented, but private mirrors or release-loader URLs should not be exposed unless intentionally public.
